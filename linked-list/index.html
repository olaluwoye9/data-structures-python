Linked List

The LinkedList class supports basic operations using singly or doubly linked nodes.

Singly Linked List

  • insertAtEnd(item)
  • insertAfter(index, item)
  • deleteItem(item)
  • update(index, item)
  • access(index)
  • search(item)
  • display()
