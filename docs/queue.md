# Queue

The `Queue` class follows **First-In-First-Out (FIFO)** behavior.

## Operations
- `enqueue(item)`
- `dequeue()`
- `peek()`
- `is_empty()`

## Example

```python
from data_structures.queue import Queue

- q = Queue()

- q.enqueue(10)

- q.enqueue(20)

- print(q.dequeue())
